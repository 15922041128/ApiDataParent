1注册功能2获取图片验证码立即测试

标示：reportRegisterCaptcha

  • 接口描述：
  • 接口地址：https://way.jd.com/intecredit/reportRegisterCaptcha
  • 请求方式：HTTPS GET POST
  • 请求示例：https://way.jd.com/intecredit/reportRegisterCaptcha?name=&userid=&appkey=您申请的APPKEY

QUERY请求参数   

名称 类型 必填 示例值 描述
name String 是 注册用户的中文姓名
userid String 是 调用核心API获取唯一标识userid，类似：001201601011410389100716

返回示例值

{
    "result": true,
    "code": 0,
    "msg":"https://api.intecredit.cn/captcha/404.jpg"
}
说明：result=true表示请求成功，result=false表示请求失败，具体错误需要判断code 
（1）当code=10006时表示请求图片验证码失败 
（2）当code=0时表示请求成功，可以从msg字段里获取图片验证码 
（3）当code=10000时表示流程请求不对，请从第一步开始 
（4）当code=10001时表示系统异常

使用时请先调用最后一个接口，获取唯一用户标识userid。

 

央行个人征信报告授权数据由3个独立的部分构成：注册、登陆授权和快捷登陆授权。调用每一部分前都要先调用相对应的获取userid接口。

（1）未在“中国人民银行征信中心系统（http://www.pbccrc.org.cn）”注册用户，须先调用注册接口进行用户注册或在“中国人民银行征信中心系统（http://www.pbccrc.org.cn）”官网注册；

（2）已注册用户但未获取央行征信报告查询码的用户，可调用登陆授权接口获取查询码；

（3）已拥有央行征信报告查询码用户可以直接授权账号、密码及授权码，提交查询请求获取查询结果。
